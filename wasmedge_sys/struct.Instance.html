Expand description

An Instance represents an instantiated module. In the instantiation process, An Instance is created from alModule. From an Instance the exported functions, tables, memories, and globals can be fetched.

A module instance is usually returned via one of the following APIs:

  • Executor
    • Executor::register_named_module (example)
    • Executor::register_active_module (example)
  • Vm
    • Vm::active_module (example)
  • Store
    • Store::module (example)
